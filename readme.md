## 🛠️ Como Executar os Códigos

### Pré-requisitos

1. Instale o Node.js (caso ainda não tenha instalado). [Baixe aqui](https://nodejs.org/).
2. Clone este repositório:
   ```bash
   git clone https://github.com/Tauan-dev/Target-desafio-backend.git
   ```
